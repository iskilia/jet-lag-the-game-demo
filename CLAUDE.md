# CLAUDE.md

## Commands
- `uv run python main.py` — run the game
- `uv run ruff check` — check for linting errors
- `uv run pytest` - run tests

## Conventions
- Never make any changes to the code unless I ask you to. 

## Hard rules
- Don't add any more train companies to the train_stations.csv file. 
- Don't change the game mechanics unless I ask you to.
- Don't make any changes to the seeker_questions.md file unless I ask you to.
- Don't change the CLAUDE.md file unless I ask you to.

## Soft rules
- Try to keep the code clean and organized.
- Every function should have a docstring explaining what it does. Make sure the docstring is written using the google style for docstrings.
- Every function cannot have more than 200 lines
